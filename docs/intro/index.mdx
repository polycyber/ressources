# Introduction

Trouvez toutes les infos pour commencer en cyber

## PolyCyber

Cette base de connaissance est une initiative de PolyCyber, société technique étudiante de Polytechnique Montréal dédiée à la cybersécurité. 

## Objectifs de ce projet

Ce projet a pour but de

* Faciliter le partage de ressources au sein de notre communauté
* Aiguiller les néophytes de la cybersécurité
* Conserver la matériel pédagogique créé par PolyCyber
* Offrir des guides techniques détaillés
